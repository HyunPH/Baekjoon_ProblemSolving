# 생각
문자열 배열을 할당한 후, ' ' 을 구분하는 %s로 입력받아서 하면...
공백과 enter를 구분 못해서 문장이 안만들어질 것 같다.
아니다. 
scanf(“%[^\n]s”, str); 대괄호를 사용한 형식으로 엔터를 제외한 모든 입력을 받겠다고 할 수 있다.

그리고 testcase가 안주어져서. 입력 받을 때마다 출력하면 될 듯 하다.

반복 할 때마다 arr[1001]배열 선언,
입력 후, 공백 포함해서 arr배열에 문자열로 char저장됨.

공백을 기준으로 단어들의 수를 새로 배열에 저장하고, 
그 배열들을 뒤집어서 각각 출력하기만 하면 될듯.

# 소감
문장 입력받을 때 enter구분해주는 것을 다시 써먹을 수 있었고,
단순히 출력만 하면 된다는 아이디어를 다시 떠올렸다.
